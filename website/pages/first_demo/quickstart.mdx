# Getting started
The All In One is a quick way to deploy the TRE-FX stack locally for testing and demo purposes. 
This page will guide you through the process of deploying it and show a simple use case example.

<Steps>
### Clone the repository
Start by cloning the All in One [repository](https://github.com/SwanseaUniversityMedical/DARE-TREFX-Environment1).
###  Run the stack
In the same directory as the docker-compose.yml run:

```bash copy
docker compose up -d
```
<Callout> Docker may need access to the file directory to mount the init.sql file.
Add the directory path to the shared paths from Docker -> Prefererences -> 
Resources -> File Sharing </Callout>

[Here](/deployment/stackinfo) is a list of the services included in the All In One
</Callout>
