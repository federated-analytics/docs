# Submitting a RO-Crate Task in the Submission Layer

## Prerequisites and Notes

Required Information:
- Your project name

Required Inputs:
- RO-Crate (zip file) containing the task

## Process

<Steps>
### Researcher logs onto the [Submission Layer](/deployment/stackinfo#tre-fx-website-submission-layer)

### Open the Project Page

Select the 'Home' page option in the top menu,
and select the 'Your Projects' tab in your home screen.

<img src="/images/TREFX_UserProjects.png" width="700"></img>

Find the project you wish to use, and click the project name.
This will open the project page for you.

<img src="/images/TREFX_ProjectPageMenu.png" width="700"></img>

### Open Crate Wizard

Select the 'Submissions' tab for the project.

<img src="/images/TREFX_ProjectSubmissions.png" width="700"></img>

Click on the blue 'Create Submission' button.
This will open a menu of wizards to use for submitting a task.
Select the 'Crate Wizard' option.

<img src="/images/TREFX_CrateWizardLink.png" width="200"></img>

### Prepare Submission

Once on the Crate Wizard page you enter the 'TES Name',
and select which TREs to submit the task to.
Select the RO-Crate zip file to upload, using the `Choose File` button.
Finally click the blue 'Submit' button.

<img src="/images/submission.png" width="300"></img>

### Submission Running

If submission is successful you will be shown the submission page.
At the bottom of this page is the Event Log,
which shows the task progress.

<img src="/images/TREFX_SubmissionRunningLog.png" width="500"></img>


### Egress Approval

Once the 'Query Processing' section shows 'Requested Egress'
then it is ready for Egress Management.
[Egress Approval](/activities/EgressManagerApprovingRelease)
should be sought from the Egress Manager.

<img src="/images/TREFX_SubmissionWaitingEgress.png" width="500"></img>

### Results Download

Once the task results have been approved by the Egress Manager 
the submission will show as complete.
To download the results click the green 'Completed' button at the bottom of the screen.

<img src="/images/TREFX_SubmissionFinished.png" width="500"></img>


</Steps>
