# Submitting a RO-Crate Task in the Submission Layer

## Prerequisites and Notes

Required Information:
- Your project name

Required Inputs:
- RO-Crate (zip file) containing the task

## Workflow

<Steps>
### Researcher logs onto the [Submission Layer](/deployment/stackinfo#tre-fx-website-submission-layer)

### Open the Project Page

Select the 'Home' page option in the top menu,
and select the 'Your Projects' tab in your home screen.

<img src="/images/TREFX_UserProjects.png" width="500"></img>

Find the project you wish to use, and click the project name.
This will open the project page for you.

<img src="/images/TREFX_ProjectPageMenu.png" width="500"></img>

### Open Crate Wizard

Select the 'Submissions' tab for the project.

<img src="/images/TREFX_ProjectSubmissions.png" width="500"></img>

Click on the blue 'Create Submission' button.
This will open a menu of wizards to use for submitting a task.
Select the 'Crate Wizard' option.

<img src="/images/TREFX_CrateWizardLink.png" width="500"></img>

### Prepare Submission

Once on the Crate Wizard page you enter the 'TES Name',
and select which TREs to submit the task to.
Select the RO-Crate zip file to upload, using the `Choose File` button.
Finally click the blue 'Submit' button.

<img src="/images/submission.png" width="500"></img>

### Submission Running

If submission is successful you will be shown the submission page.
You will need to wait for the task to finish,
and then for the Egress Manager to approve the data release.


</Steps>
